Ce fichier explique comment Visual Studio a créé le projet.

Les étapes suivantes ont été utilisées pour générer ce projet :
- Créez un projet d’API web ASP\.NET Core.
- Mettez à jour le fichier projet pour ajouter une référence au projet front-end et définir les propriétés de SPA.
- Mettez à jour `launchSettings.json` pour inscrire le proxy SPA en tant qu’assembly de démarrage.
- Ajoutez le projet à la liste des projets de démarrage.
- Écrivez ce fichier.
